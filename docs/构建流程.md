## 编译前准备

请使用如下版本的mingw:

![](assets/make_package_image.png)

请配置cmake编译选项：

![](assets/make_package_image-1.png)

---
配置`CMakeLists.txt`文件，确保：

```txt
# 配置选项
set(D_OR_R OFF)                # ON: Debug, OFF: Release
set(CONSOLE OFF)               # ON: 显示控制台, OFF: 隐藏控制台（GUI模式）
```

编辑`src\main.cpp`，确保：

```cpp
#define DEBUG false
```

以便将控制台输出到日志文件

## 编译

清理缓存，如有需要，可提前删除`cmake-build-release`等类似目录

![](assets/make_package_image-2.png)

点击编译按钮，等待编译完成
