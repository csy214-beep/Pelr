核心特性

一切为了更好的桌面体验

Pelr 将 Live2D 动画、AI 对话与实用工具融为一体，打造独一无二的桌面伴侣。

Live2D 虚拟角色

支持 model3.json 格式的 Live2D 模型，生动灵动的桌面伴侣，支持丰富的表情与动作切换。

智能 AI 对话

支持 OpenAI 兼容的 AI 服务，自然语言交互，让桌面角色真正「活」起来。

语音合成

内置 OpenAI-Edge-TTS 与讯飞 TTS 服务，高质量的语音反馈，让交互更加自然。

TODO 功能

简洁的待办系统，添加事件并设置提醒，桌面伙伴帮你记住每一件小事。

启动管理

可视化管理应用程序，一键启动 Windows 上的文件与链接，继承自 QuickTray。

键盘监听

实时显示按键状态，继承自 KeyMonitor，让每一次敲击都清晰可见。

天气服务

集成 OpenWeather，实时获取天气信息，桌面伙伴随时告知你天气变化。

音乐托盘

托盘图标随系统音量旋转，继承自 Rotating Rhythm，让系统托盘也有了生气。

高度可定制

丰富的设置选项，从模型到 TTS、从快捷键到主题，满足你的个性化需求。

技术栈

坚实的技术基石

Pelr 采用 C++ 核心与 Python 工具链，兼顾性能与灵活性。

C++ 核心 高性能引擎
Qt 5.15 跨平台框架
OpenGL 图形渲染
Live2D Cubism 2D 动画引擎
Python 3.11 TTS 服务端
Edge-TTS 语音合成
